American Enterprise Project

Overview: Reports must promote an awareness of some part of the American (free) enterprise system with teh school and/or community and aswer the Report Format questions. The project is intended to help members learn more about the free enterprise system and is designed for chapter participation.

Community Service Project

Overview: Reports must describe a chapter project that serves the community and conforms to the guidelines and format stated. The activity must be in the interest of the local community, focus on one project, and be designed for chapter participation.

Local Chapter Activities Report

Overview: The report should summarize the activities of the local chapter between the start of the previous State Leadership Conference and the current State Leadership Conference. This report meets one of the requirements for the Outstanding Chapter Award of Merit program.
